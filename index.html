Settings

Signal K Server

Point this at your Signal K server. On a Cerbo GX running Venus OS Large the default is the Cerbo's IP on port 3000. On OpenPlotter / Raspberry Pi it is likewise port 3000.

Display Units
Connectivity

Pauses the weather-radar animation and its background refresh whenever the Weather tab isn't open, and shows a single static radar frame instead of looping. Recommended on metered cellular connections. The dashboard, electrical and LAN camera data use no internet.

Engines

Each engine maps to the Signal K path propulsion.<id>.*. Twin-engine boats commonly use port and starboard. Leave an engine's id blank to hide it. Check your server's Data Browser if unsure.

Generator

Reads electrical.generators.<id>.*. Nominal voltage and frequency set the out-of-range warning bands. Leave the id blank to hide the generator panel.

Electrical (12 VDC / 240 VAC)

12 VDC load is calculated from the SmartShunt (electrical.batteries.<id>.*) as bus voltage × discharge current. 240 VAC loads read from electrical.ac.consumption.*.

Cameras (up to 4)

Browsers can play MJPEG, JPEG snapshot, HLS (.m3u8) and plain video directly. RTSP is not playable in a browser — route it through a gateway (go2rtc / MediaMTX / Frigate) that outputs HLS or MJPEG. Leave a URL blank to hide that camera; use Auto to detect the type from the URL.
